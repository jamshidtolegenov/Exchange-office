# Обменный пункт — веб-приложение

React-приложение для автоматизации работы сети обменных пунктов (4 города: Шымкент, Алматы, Москва, Ташкент).

## Быстрый старт

```bash
npm install
npm start
```

Откроется http://localhost:3000

## Сборка для продакшна

```bash
npm run build
```

## Функциональность

### Оператор города
- Просмотр остатков своей кассы
- Покупка валюты у клиента (клиент отдаёт USD/RUB/UZS → получает KZT)
- Продажа валюты клиенту (клиент отдаёт KZT → получает USD/RUB/UZS)
- Мгновенный пересчёт результата при вводе суммы
- Защита от отрицательного остатка
- Журнал операций своего пункта

### Администратор
- Сводный баланс по всем 4 городам
- Таблица балансов по городам с итогами
- Изменение курсов валют
- Журнал всех операций с фильтром по городу и дате
- Экспорт журнала в CSV
- Сброс данных к начальным значениям

## Хранение данных

Данные хранятся в `localStorage` браузера:
- `exch_balances` — остатки по городам
- `exch_rates` — текущие курсы валют
- `exch_ops` — журнал всех операций
- `exch_user` — текущая сессия

## Переход на реальный бэкенд

Все обращения к данным инкапсулированы в `DataContext.jsx`.  
Замените функции `load()` / `save()` / `doOperation()` на вызовы API:

```js
// Пример замены:
const doOperation = async ({ city, opType, currency, amount, operator }) => {
  const res = await fetch('/api/operations/exchange', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${token}` },
    body: JSON.stringify({ city, opType, currency, amount, operator }),
  });
  return res.json();
};
```
